# 2026-03-11 GLM (Kimi) 智谱清言网页版自动化研究

## 研究目标
使用 Browser Relay 自动化 Kimi (智谱清言) 网页版的对话

## 测试结果

### 1. 网站 URL
- **chatglm.cn** → 重定向到 **kimi.com**
- 主站: https://kimi.com/ 或 https://www.kimi.com/

### 2. 登录状态
- **需要登录**才能使用聊天功能
- 登录弹窗会在发送消息时出现，阻止自动化

### 3. 页面结构 (已知的 CSS 选择器)
- 输入框: `textbox` (ref=e101)
- 发送按钮: 输入框旁边的图片按钮 (ref=e107)
- 新建会话链接: `link "新建会话 ⌘ K"` (ref=e9)

### 4. 尝试的操作
1. 打开 https://kimi.com/ - 成功，显示登录弹窗
2. 导航到 /?chat_enter_method=new_chat - 暂时关闭登录弹窗
3. 输入消息: "你好，请用一句话介绍你自己"
4. 按 Enter 发送 - 登录弹窗再次出现，阻止响应

### 5. 结论
- **网页版需要登录**，无法在未登录状态下自动化
- 需要先通过手机号或微信登录获得会话 Cookie
- 登录后可使用 Browser Relay 自动化

### 6. 下一步建议
1. 用户需要先登录 Kimi 账号
2. 登录后可尝试通过 Browser Relay 自动化
3. 或者考虑使用 Kimi API (kimi-api) 进行无头对话
