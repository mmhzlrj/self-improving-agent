## 经验教训（重要！已记录到 .learnings/LEARNINGS.md）

### 失败的尝试
1. 直接 API 调用 - 需要 a_bogus 签名 ❌
2. osascript 激活窗口 - 窗口会跳动，打扰用户 ❌
3. JS 键盘事件模拟 - 被豆包反自动化检测拦截 ❌
4. DOM click() 简单调用 - React 组件不响应 ❌

### 成功方案
- 使用 `openclaw browser evaluate` 执行 JavaScript（零窗口跳动）
- CSS 选择器找到输入框：`document.querySelector('textarea')`
- 遍历父容器找到发送按钮（带 SVG 的按钮）
- 通过页面快照动态获取模型切换按钮的 ref

### 关键代码
```javascript
// 找到发送按钮并点击
var parent = textarea.parentElement;
while (parent && parent.tagName !== 'BODY') {
    var btns = parent.querySelectorAll('button');
    for (var i = btns.length - 1; i >= 0; i--) {
        if (btns[i].querySelector('svg')) {
            btns[i].click();
            return 'sent';
        }
    }
    parent = parent.parentElement;
}
```
