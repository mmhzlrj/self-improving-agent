# 复制按钮检测脚本调试记录

## 问题背景
用户要求检测每个 AI 平台网页的复制/分享/编辑按钮是否出现。

## 检测脚本（最终版）

```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.connect_over_cdp("http://127.0.0.1:18800")
    
    platforms = [
        ("deepseek", "DeepSeek", "已思考"),
        ("chatglm", "智谱", "思考结束"),
        ("qwen", "千问", "已经完成思考"),
        ("doubao", "豆包", "已思考"),
        ("kimi", "Kimi", "编辑")
    ]
    
    for page in browser.contexts[0].pages:
        url = page.url
        
        for key, name, keyword in platforms:
            if key in url:
                print(f"\n=== {name} ===")
                
                # 1. 滚动到关键词位置
                page.evaluate(f"""
                    () => {{
                        const all = document.querySelectorAll('*');
                        for (let el of all) {{
                            if (el.innerText && el.innerText.includes('{keyword}')) {{
                                el.scrollIntoView({{ behavior: 'instant', block: 'center' }});
                                return;
                            }}
                        }}
                    }}
                """)
                
                # 2. 等待网络稳定
                page.wait_for_load_state("networkidle")
                page.wait_for_timeout(500)
                
                # 3. hover 触发按钮显示
                page.evaluate(f"""
                    () => {{
                        const all = document.querySelectorAll('*');
                        for (let el of all) {{
                            if (el.innerText && el.innerText.includes('{keyword}')) {{
                                if (el.hover) el.hover();
                                else if (el.dispatchEvent) {{
                                    el.dispatchEvent(new MouseEvent('mouseenter', {{ bubbles: true }}));
                                }}
                                return;
                            }}
                        }}
                    }}
                """)
                
                # 4. 等待按钮显示
                page.wait_for_timeout(1500)
                
                # 5. 找所有可见元素中包含"复制"/"分享"/"编辑"的
                result = page.evaluate("""
                    () => {
                        const result = { copy: 0, share: 0, edit: 0, elements: [] };
                        
                        // 遍历所有可见元素
                        document.querySelectorAll('*').forEach(el => {
                            if (!el.offsetParent) return; // 不可见
                            
                            const rect = el.getBoundingClientRect();
                            if (rect.width === 0 || rect.height === 0) return; // 无尺寸
                            
                            const text = (el.innerText || '').trim();
                            
                            // 只统计正好是"复制"/"分享"/"编辑"的元素
                            if (text === '复制') { result.copy++; result.elements.push({tag: el.tagName, class: el.className.substring(0,30)}); }
                            if (text === '分享') { result.share++; result.elements.push({tag: el.tagName, class: el.className.substring(0,30)}); }
                            if (text === '编辑') { result.edit++; result.elements.push({tag: el.tagName, class: el.className.substring(0,30)}); }
                        });
                        
                        return result;
                    }
                """)
                
                print(f"复制={result['copy']}, 分享={result['share']}, 编辑={result['edit']}")
                
                # 打印找到的元素详情
                for e in result['elements'][:10]:
                    print(f"  {e['tag']}: {e['class']}")
                
                break
```

## 发现的问题

### 问题1：选择器不对
- 之前只检测 `button` 标签
- 实际上 Kimi 的按钮是 `DIV` 标签，class 是 `simple-button`

### 问题2：页面内容已变化
- 之前的测试对话内容（"一句话解释人工智能"）已被覆盖
- 现在页面内容变成了代码讨论

### 问题3：检测范围太大
- `get_by_text("复制")` 会匹配页面所有包含"复制"的文字
- 需要限定在**回复附近**查找

## 修正方法

### 方法1：精确匹配文本
```python
# 只统计正好是"复制"/"分享"/"编辑"的元素
if (text === '复制') { result.copy++; }
if (text === '分享') { result.share++; }
if (text === '编辑') { result.edit++; }
```

### 方法2：扩大选择器范围
```python
# 包含 DIV
document.querySelectorAll('button, div, span, [role="button"]')
```

### 方法3：限定在关键词附近
```python
# 距离关键词 300px 以内
if (distY < 300 && distX < 400) { ... }
```

## 经验总结

1. **hover() 是关键**：很多按钮需要悬停才会显示
2. **选择器要全面**：不能用 `button` 就够了，还要包含 `div, span`
3. **精确匹配**：用 `text === '复制'` 而不是 `text.includes('复制')`
4. **限定范围**：在关键词附近查找，避免匹配到页面其他位置的文字
