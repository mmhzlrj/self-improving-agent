# H2C + 闭环步进升级 Cyber Bricks 精准抓取方案

> 调研时间：2026-03-31
> 背景：Cyber Bricks 原版开环舵机精度不足，评估 H2C 打印 + 步进闭环 + 视觉反馈的落地方案

---

## 一、H2C 打印能力确认

### 拓竹 H2C 关键规格

| 指标 | H2C 规格 | 说明 |
|------|---------|------|
| XY 定位精度 | **±50μm** | 核心精度指标 |
| Z轴定位精度 | ±50μm | 层间精度 |
| 最小层厚 | 0.08mm（80μm）| 比这更薄容易堵料 |
| 最小细节 | 0.1~0.2mm | 超过即可打印 |
| 可打印最小齿轮模数 | **模数 0.2~0.3** | 临界值，50μm精度刚好够 |

### 打印材料选择

| 材料 | 价格/kg | 打印难度 | 力学性能 | 推荐度 |
|------|--------|---------|---------|--------|
| **PETG** | ~$25 | 简单 | 韧性好、强度高、耐冲击 | ⭐⭐⭐⭐⭐ **首选** |
| PLA | ~$20 | 最简单 | 一般，易脆 | ⭐⭐⭐ 外观件 |
| ABS | ~$25 | 难（需热箱）| 好 | ⭐ 不推荐 |
| PA（尼龙）| ~$50 | 难 | 很强 | ⭐⭐ 高强度件 |

**结论：PETG 性价比最高**——价格比 PLA 贵 20%，但韧性/强度好很多，适合做关节支架/齿轮/夹爪。

## 三、零件清单

### H2C 打印件（PETG）

| 零件 | 精度要求 | 说明 |
|------|---------|------|
| 主齿轮（模数0.3，10:1减速）| 50μm | 齿轮精度决定减速箱性能 |
| 行星齿轮组（模数0.3）| 50μm | 10:1或20:1减速 |
| 输出轴轴承座 | 50μm | 保证同心度 |
| 夹爪安装座 | 100μm | 略低可接受 |
| 夹爪手指（两指）| 100μm | 配合面精铣 |
| 电机安装支架 | 200μm | 定位用，要求低 |

> ⚠️ 齿轮部分：模数0.3是H2C打印精度的临界值，建议齿轮买金属成品（成本低+可靠），行星减速箱买现成的 10:1 金属行星箱，H2C 只打印壳体和连接件。

### 外购零件（每轴）

| 零件 | 推荐型号 | 价格/轴 | 说明 |
|------|---------|---------|------|
| 步进电机 | 42步进（NEMA 17，1.8°/步）| ~$8 | 便宜可靠 |
| 行星减速箱 | 10:1 金属行星减速箱（输入5mm轴）| ~$12 | 买金属的，H2C打印精度临界 |
| 磁编码器 | AS5600 | ~$2 | I2C接口，12-bit，分辨率0.087°/LSB |
| 驱动板 | TMC2208（静音驱动）| ~$3 | 细分控制（16细分=3200步/转）|
| 柔性联轴器 | 5mm to 5mm | ~$1 | 连接电机轴和减速箱输入轴 |
| 夹爪手指 | PETG打印 + 橡胶垫 | ~$1 | 自制 |

**每轴总成本：~$27**（一臂两轴 ≈ $54，一台 Cyber Bricks 完全升级 ≈ $108）

## 四、双闭环控制原理

```
目标角度 θ_target
    │
    ▼
┌───────────────┐
│  外环：位置闭环 │
│  AS5600 读数   │ ← 磁编码器（12-bit，0.087°/LSB）
│  误差 = θ_err │
└───────┬───────┘
        │
        ▼
┌───────────────┐
│  内环：电流闭环 │
│  TMC2208 驱动  │ ← 细分控制（16细分 = 3200步/转）
│  检测堵转     │
└───────┬───────┘
        │
        ▼
┌───────────────┐
│  步进电机输出  │
│  42mm + 10:1  │
│  = 32000 步/转 │
│  末端 = 0.011°/步│
└───────────────┘
```

### 精度计算

| 参数 | 数值 |
|------|------|
| 电机步距角 | 1.8°/步 |
| TMC2208 细分 | 16细分 → 0.1125°/细分 |
| 行星减速比 | 10:1 → 0.01125°/细分 |
| 夹爪臂长 | 100mm |
| **末端分辨率** | 0.01125° × π/180 × 100mm = **0.0196mm/细分** |

加上 AS5600 反馈校正（12-bit = 4096点/转）→ 实际闭环精度 **±0.1mm**（甚至可达 ±0.05mm）

## 五、与 Jetson Nano + ESP32-Cam 联动

| 任务 | 执行者 | 方式 |
|------|--------|------|
| 物体识别定位 | **ESP32-Cam**（YOLOv8n）| RTSP 流，实时检测物体坐标 |
| 末端位置反馈 | **AS5600**（精度 0.087°）| I2C 读到 ESP32-C3，转发给 Jetson |
| 误差校正 | **Jetson Nano** | 对比目标位置 vs AS5600 实际读数，补偿 |
| 逆运动学求解 | **Jetson Nano** | 解算各关节目标角度 |
| PID 控制输出 | **ESP32-C3** | 执行 PID，向 TMC2208 发细分脉冲 |
| 夹爪开合 | **ESP32-C3** | PWM 控制 |

## 六、与原版 Cyber Bricks 对比

| 对比项 | 原版 Cyber Bricks | 升级版 |
|--------|------------------|--------|
| 关节反馈 | ❌ 无 | ✅ AS5600 磁编码器 |
| 控制精度 | 开环，PWM | 双闭环 PID |
| 夹爪 | 平行杆（PLA）| PETG 定制 + 橡胶垫 |
| 末端精度 | ±2~5mm | **±0.1mm** |
| 抓取尺寸 | 大物体（>5cm）| **1-10cm** |
| 成本/轴 | $0（自带）| **+$27** |

## 七、细分物体抓取能力

| 物体尺寸 | 原版 Cyber Bricks | 升级版 | 说明 |
|---------|------------------|--------|------|
| < 1cm（螺丝/针）| ❌ 不行 | ⚠️ 勉强（需调参）| H2C齿轮模数临界，末端±0.1mm刚好够 |
| 1-5cm（零件）| ❌ 不行 | ✅ 可行 | 目标精度，稳了 |
| 5-10cm（工具）| ⚠️ 勉强 | ✅ 轻松 | 加误差余量，更可靠 |
| > 10cm | ✅ 可以 | ✅ 最佳 | 原版就能做，升级后更准 |
