
# Browser Relay 排查计划（豆包提供）

## 下一步排查步骤

### 步骤1：获取 Browser Relay 扩展的调试日志（最高优先级）

需要用户配合：

1. **打开 Chrome 扩展调试面板**：
   - 进入 `chrome://extensions/`
   - 开启「开发者模式」（右上角）
   - 找到「Openclaw Browser Relay」扩展
   - 点击「检查视图」→「背景页」（或「Service Worker」）

2. **切换到 Console 标签，查看日志**

3. **复现问题并捕获日志**：
   - 执行获取标签页 curl 命令
   - 执行创建标签页 curl 命令
   - 记录 Console 中的日志

### 步骤2：排查扩展的 CDP 转发逻辑

检查扩展代码中的：
- 标签页列表返回逻辑
- 创建标签页接口转发逻辑
