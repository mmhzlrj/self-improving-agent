## 一、微软 Project Silica — 最新数据（截至2026年3月）

### 1.1 核心技术突破（2026年2月《Nature》论文）

微软研究院于 **2026年2月18日** 在《Nature》期刊发表论文 *Laser writing in glass for dense, fast and efficient archival data storage*，公布 Project Silica 最新里程碑数据：

| 参数 | 数据 | 说明 |
|------|------|------|
| **存储介质尺寸** | 120 mm × 120 mm × 2 mm | 约等于杯垫大小 |
| **最大层数** | **301 层** | 垂直堆叠实现高密度 |
| **单片容量** | **4.84 TB**（熔融石英）<br>**≈2.02 TB**（硼硅酸盐玻璃） | 硼硅玻璃成本更低，更适合商业化 |
| **存储密度** | 1.59 Gbit/mm³ | |
| **写入速度** | 单光束：**25.6 Mbit/s**<br>4光束并行：**65.9 Mbit/s** | 填满4.84TB需超过150小时（主要瓶颈） |
| **写入能效** | 8.85 – 10.1 nJ/bit | |
| **读取方式** | 光学显微镜 + CNN机器学习 + LDPC纠错 | |
| **保存寿命** | **≥10,000年** | 加速老化实验：290°C高温验证，室温更长 |
| **介质扩展** | 从熔融石英扩展至低成本**硼硅酸盐玻璃** | 商业化关键突破 |

**数据来源：**
- 微软研究院官方博客：Project Silica advances
- 《Nature》2026年2月18日论文
- 麻省理工科技评论（MIT Tech Review China）报道

### 1.2 读写速度详解（当前瓶颈）

- **写入**：单束飞秒激光 25.6 Mbit/s ≈ **3.2 MB/s**；4束并行约 8.2 MB/s
- **读取**：需光学显微镜逐层扫描 + AI解码，速度尚未公开但预计更慢
- **对比**：典型机械硬盘写入约 100-200 MB/s，固态硬盘 500-3000 MB/s
- **结论**：玻璃存储写入速度比机械硬盘慢 **数十倍**，目前仅适合"写入一次、长期读取"的冷归档场景

### 1.3 商业化路线图

| 阶段 | 时间 | 内容 |
|------|------|------|
| **试点期** | 2025–2027年 | 特定领域：政府档案、科研数据、娱乐母带存储 |
| **企业可用** | 2027–2030年 | 面向企业数据归档场景商业供应 |
| **广泛采用** | 2030–2035年 | 数据中心大规模部署 |

**主要挑战：**
- 写入速度慢（核心瓶颈）
- 飞秒激光系统成本高（单套系统数十万美元）
- 读取设备尚未小型化

## 四、技术定位与市场分析

### 4.1 玻璃存储的定位

**目标场景：** 冷数据归档（写入一次，长期读取，存放数十年以上）
- 替代对象：磁带（Tape）、蓝光光盘库（PDD/UHD）
- 不适用：热数据（需要频繁读写）存储

**市场空间：**
- 全球数据量爆发式增长（ZB量级），冷存储需求巨大
- AI大模型训练数据激增，长期归档需求旺盛
- 国家档案法对历史数据长期保存的要求

### 4.2 玻璃存储 vs 主流存储介质对比

| 介质 | 典型容量 | 寿命 | 写入次数 | 单位成本 | 适用场景 |
|------|---------|------|---------|---------|---------|
| **玻璃光学存储** | 360 TB/片 | 万年+ | Write-once | 中（量产前） | 冷归档 |
| **蓝光光盘（PDD）** | 300 GB/片 | ~50年 | Write-once | 低 | 冷归档 |
| **磁带（LTO-9）** | 18 TB/盘 | ~30年 | 多次 | 很低 | 冷归档 |
| **机械硬盘（HDD）** | 24 TB/盘 | ~5年 | 多次 | 低 | 温/热数据 |
| **固态硬盘（SSD）** | 61 TB/盘 | ~10年 | 多次 | 高 | 热数据 |

## 五、总结

1. **微软 Project Silica** 已通过2026年Nature论文证明技术可行性（301层/4.84TB/万年寿命），但商业化仍需等待（2030年前后），写入速度是核心瓶颈。

2. **中国（华中科技大学/一尧科技）** 在玻璃光学存储领域已实现"弯道超车"：单盘容量360TB（是微软的75倍）、2025年底量产、商业化领先约2年、全国产化自主可控。

3. **差距本质**：微软起步早（2018年启动），但中国团队有后发优势 + 工程化能力强 + 政策支持。核心差距不在学术前沿，而在飞秒激光器等关键设备的成熟度和量产成本控制。

4. **关键风险**：国产的"360TB"数据尚未经第三方独立验证；微软的耐久性与寿命数据有Nature论文背书更为扎实；玻璃存储市场教育尚需时间。
