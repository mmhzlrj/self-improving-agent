### 技术参数

- **硬件**: Ubuntu RTX 2060 6GB (VRAM 5.5GB 可用)
- **分辨率**: 256×144
- **帧数**: 10帧 (0.4秒)
- **生成速度**: ~5.2秒/帧
- **总耗时**: ~12分钟（6条视频）

### 遇到的问题
1. qwen_inference_server 占用 1GB VRAM → 禁用后解决
2. SVD 需要 6GB VRAM，RTX 2060 只有 5.6GB → 使用 CPU offload 解决
3. HuggingFace 访问超时 → 使用 HF Mirror (hf-mirror.com)
4. MiniMax 图像/视频 API 不可用

### 文件位置
- Ubuntu: /home/jet/视频/
- MacBook: ~/Movies/0-1-videos/

### 经验
- SVD VRAM 需求比预期高，需要完整 6GB
- CPU offload 可以降低 VRAM 需求但速度更慢
- 每条视频 ~2分18秒，生成速度约 0.07fps

- `--load-extension` 在全新 profile 上能加载扩展，但 Chrome 146 会自动禁用（`state=None`）
