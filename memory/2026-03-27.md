### 遇到的问题及解决
1. **SECURITY ERROR: Cannot connect over plaintext ws://** → 加 `OPENCLAW_ALLOW_INSECURE_PRIVATE_WS=1`
2. **gateway token missing** → 加 `OPENCLAW_GATEWAY_TOKEN=xxx`

- `openclaw config set gateway.bind lan` 成功写入配置

- `openclaw devices approve --url ws://192.168.1.13:18789 --token xxx` + `OPENCLAW_ALLOW_INSECURE_PRIVATE_WS=1` → 安全检查通过但 gateway timeout
- Gateway 确实在 `*:18789` 监听（lsof 确认），但 CLI 连接超时
