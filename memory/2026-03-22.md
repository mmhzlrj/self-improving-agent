### 任务记录规范（必须遵守）

**问题：** 收到任务后直接执行，没有记录。导致任务内容丢失，无法追溯。

**正确流程：**
1. 收到任务时 → 立即在当天 yyyy-mm-dd.md 记录
   ```
   hh-mm 收到任务：<任务内容>
   ```
2. 完成任务后 → 更新记录加上完成时间和总结
   ```
   hh-mm 完成：<内容总结>
   ```

**注意：** 先记录再执行，不要先执行后记录。
