### 豆包自动化输入技术突破（2026-03-09）
- 问题：豆包页面使用 React 动态渲染，aria-ref 不稳定，导致输入超时
- 解决：使用 JavaScript 注入直接操作 DOM
- 方法：
  1. `document.querySelector('div[role="textbox"]')?.focus()` 聚焦
  2. `document.execCommand('insertText', false, '文字')` 输入
- 关键：不依赖 aria-ref，直接操作 DOM
- 测试：成功发送"你好，测试自动化输入"，豆包正常回复

### 微信公众号文章获取技术突破（2026-03-09）
- 问题：微信公众号文章有验证码保护，curl 直接请求被拦截
- 失败尝试：
  1. curl 直接请求 → 返回"环境异常"验证页面
  2. jina.ai 服务 → 请求超时
  3. 浏览器直接访问 → 未测试
  
- 成功方法：使用 `openclaw browser open` 命令
  - 命令：`openclaw browser open "https://mp.weixin.qq.com/s/xxx"`
  - 原理：OpenClaw 自带浏览器可以绕过验证（或用户已登录）
  - 获取内容：`openclaw browser snapshot --json`

- 关键要点：
  1. 微信公众号文章需要登录态，OpenClaw Browser 可以保持登录
  2. 不要用 curl/网络请求工具直接抓取
  3. 优先使用 OpenClaw 自带浏览器

### Browser Relay vs OpenClaw Browser 的区别
- Browser Relay：控制用户已有的 Chrome 浏览器（需要扩展）
- OpenClaw Browser：OpenClaw 自带的专用浏览器（命令：`openclaw browser open`）
