# 2026-03-17 智库平台测试记录

## 第1次测试

### 测试问题
```
请用一句话解释什么是人工智能？
```

### 测试结果

| 平台 | 发送 | 获取回复 | 实际内容 |
|------|------|----------|----------|
| DeepSeek | ✅ | ✅ | 新问题回答 |
| 智谱 | ✅ | ✅ | 历史内容 |
| 豆包 | ✅ | ✅ | 历史内容 |
| Kimi | ❌ | ✅ | 历史内容 |
| 千问 | ❌ | ✅ | 历史内容 |

### 问题分析

1. **Kimi 发送失败**
   - 错误: `Timeout 30000ms exceeded waiting for locator("textarea")`
   - 原因: Kimi 页面结构可能不同，需要使用不同的选择器

2. **千问发送失败**
   - 错误: `Timeout 30000ms exceeded waiting for locator("textarea")`
   - 原因: 千问页面有代码编辑器阻挡输入框

3. **获取到历史内容**
   - 原因: 页面有之前的问题对话，hook 检测到内容长度>200就认为完成
   - 问题: 没有验证是否是最新问题的回答

### 需要改进

1. **发送前清空页面或创建新会话**
2. **验证回复是否包含新问题关键词**
3. **修复 Kimi 和千问的选择器**

### 成功获取新回答的平台

- **DeepSeek**: "我们被要求"用一句话解释什么是人工智能"。这个问题很简单直接..."
