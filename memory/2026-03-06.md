### 1. QQ 机器人配置成功 ✅

**配置流程：**
1. 安装插件：`openclaw plugins install @sliverp/qqbot@latest`
2. 添加配置：`openclaw channels add --channel qqbot --token "APPID:AppSecret"`
3. 重启 Gateway：`openclaw gateway restart`

**关键教训：**
- OCR 识别容易把字母 **O** 和数字 **0** 混淆
- 识别结果：`KU03...` → 正确：`KUO3...`
- **建议**：配置完成后务必核对原始图片中的字符

### 2. OCR 本地化方案 ✅

**问题**：不想用外部 API（隐私顾虑）

**解决方案**：使用 macOS Vision 框架（纯本地）

```swift
import Foundation
import Vision
import AppKit

let request = VNRecognizeTextRequest()
request.recognitionLevel = .accurate
# 调用系统 API，完全离线
```

**首次运行**：约1分钟（编译标准库）
**后续运行**：秒级
