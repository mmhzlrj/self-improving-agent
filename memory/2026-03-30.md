#### 关键教训
1. **act数据有NaN**：1.09%的NaN值导致训练失败 → 用`np.nan_to_num`修复
2. **学习率**：1e-4太高 → 3e-6才稳定
3. **SIGReg简化**：原始SIGReg内存爆炸 → 用方差正则化替代
4. **帧索引映射**：parquet row index ≠ video frame index
5. **AV1解码**：Ubuntu不支持AV1硬件解码 → OpenCV软解慢
