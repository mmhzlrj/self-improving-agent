# Kimi 网页版自动化研究 - 2026-03-11

## 研究结果

### 1. 打开方式
- URL: https://kimi.moonshot.cn/
- 命令: `openclaw browser open https://kimi.moonshot.cn/`

### 2. 登录状态
- **需要登录**：Kimi 需要手机号+验证码登录
- 登录界面元素：
  - textbox "+86" [ref=e14]
  - textbox "请输入手机号" [ref=e15]
  - textbox "请输入验证码" [ref=e16]
  - button "发送验证码" [ref=e17]
  - button "登录" [ref=e18]

### 3. 已登录状态下的元素
当登录后，页面结构：
- link "新建会话 ⌘ K" [ref=e2]
- textbox [ref=e10] - 主输入框（"有什么我能帮您的吗？"）

### 4. 自动化流程

**已登录状态：**
```bash
# 1. 打开 Kimi
openclaw browser open https://kimi.moonshot.cn/

# 2. 获取快照确认元素
openclaw browser snapshot --efficient

# 3. 输入消息（输入框 ref=e10）
openclaw browser type e10 "你好，请用一句话介绍你自己"

# 4. 发送（按 Enter）
openclaw browser press Enter

# 5. 等待回答（5-10秒）
sleep 10

# 6. 获取回答
openclaw browser snapshot --efficient
```

**未登录状态：**
- 需要先完成手机号登录
- 需要用户提供手机号和验证码
