## 17:41 mdview.py 三层 bug 修复

### 问题
Minimax subagent 调用 mdview.py 打开 markdown 文件，浏览器实际停在 dashboard.html。

### 三层根因
1. `webbrowser.open()` 默认 `new=0` → 激活已有 tab 而非导航
2. mdview 和 dashboard 共用 18999 端口 → 18999 被 dashboard 劫持
3. 复用检查只认进程名不认端口 → 用了错误的端口打开 URL

### 修复
- `PORT = 18999` → `18990`（独立端口）
- `webbrowser.open(url)` → `webbrowser.open(url, new=2)`（两处）
- 复用检查改为 `socket.connect_ex()` 确认端口监听
