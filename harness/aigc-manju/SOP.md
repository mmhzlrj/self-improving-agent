# AIGC 漫剧多智能体工业化生产 SOP v0.1

## 系统架构

```
OpenClaw 调度中枢
    ↓
编剧 Agent → 绘画 Agent → Prompt Agent → 文生图 Agent → 视频合成 Agent → 抽卡 Agent → 人工审核 → 推流
```

## 各环节工具选型

| 环节 | 工具 | 状态 |
|------|------|------|
| 编剧 | 豆包网页版 | ✅ 完成 |
| 绘画 | 豆包图像生成 (Seedream 4.5) | ✅ 完成 |
| 视频生成 | 豆包 Seedance 2.0 | 🔄 进行中 |
| 抽卡 | 豆包 | 待测试 |
| 推流 | 抖音API / Browser Relay | 待测试 |
