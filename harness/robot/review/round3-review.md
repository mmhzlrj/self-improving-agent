**原文推荐**：Prophesee 事件相机 / RealSense D455f
**用户拒绝原因**：0-1 的「0」本身就是视觉方案；被错误修改；iPhone 16 Pro 是很好的补充

**0-1 原始视觉方案（来自六问六答）**：

> 「0」有多个摄像头组成的视觉系统，像动漫角色眼睛里的小圆点，各司其职

| 组件 | 功能 |
|------|------|
| 主眼 | 人脸识别、日常视觉 |
| 广角 | 室内场景感知、导航 |
| ToF 摄像头 | 手势识别、近距离测距 |

### 5. 通信技术

**用户采纳（附详细更正）**：

> UWB FiRa 规范核心正式版本应为 **FiRa Core 3.0**（非2.0）
> Matter 标准应为 **Matter 1.5**（2025年11月正式发布）

**Matter 1.5 关键更新**：
- 原生支持智能摄像头（基于 WebRTC，含双向语音、PTZ控制、运动检测）
- 统一闭合设备（窗帘/车库门/幕帘）
- 土壤传感器（园艺/灌溉联动）
- 能源管理（分时电价/光伏/储能）
- SDK 开源 Apache 2.0，免版税商用；产品上市需 CSA 认证付费
- 截至2025年10月认证设备超8400款；三星首个支持，苹果/谷歌/亚马逊已启动适配

| 技术 | 规格 | 备注 |
|------|------|------|
| **星闪 SLE** | 12Mbps / 20μs / 比蓝牙降60% | 替代CAN总线，0-1核心控制 |
| **星闪 SLB** | 1.2Gbps | 高速模式 |
| **UWB FiRa Core 3.0** | 27Mbps / <100mW | 室内精准跟随 |
| **Matter 1.5** | mesh / <100ms | 智能家居互联；**原生摄像头支持** |
| **Wi-Fi 7** | 30Gbps / <1ms | 视频回传 |

### 7. 具身智能 VLA 模型

**原文推荐**：π0 / AgiBot World / Helix
**用户拒绝原因**：需要解决长时序任务、动态环境适配、空间幻觉；优先选 RynnBrain，搭配下游 VLA 模型

**RynnBrain 定位**：
- 视觉-语言时空推理（CoP）
- 输出结构化坐标，非动作执行
- 记忆增强：100步任务成功率88%
- 零样本新家庭适配

**修订后推荐架构**：

```
0-1 大脑架构（RynnBrain + 下游VLA）：
├─ 贵庚（记忆 + 对话）
├─ RynnBrain-30B MoE（任务规划 + 物体记忆时空推理）
├─ RynnBrain-Nav（室内导航）
├─ RynnBrain-Plan（操作序列生成）
└─ 下游VLA（如π0/AgiBot）负责动作执行控制
```

RynnBrain 负责感知推理，下游 VLA 负责执行，**感知大模型+执行小模型**是正确架构方向。
