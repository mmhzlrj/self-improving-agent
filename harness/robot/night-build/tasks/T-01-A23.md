## 任务目标

从历史 Task 归纳数据中，自动提炼出可复用的技能模板：
1. 定时扫描 tasks 积累数据
2. Embedding 聚类（相似 tasks → 一组）
3. 每组积累 3-5 个样本后，调用 LLM 提炼为技能模板
4. 存入 skills 库
5. 新 session 可查询/建议相关技能
