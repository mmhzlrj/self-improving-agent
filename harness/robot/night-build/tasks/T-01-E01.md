## Prompt

执行以下命令，提取文件中所有标题：

```bash
grep -n '^#' harness/robot/night-build/reference/ROBOT-SOP.md | python3 -c "
import json, sys, re
results = []
for line in sys.stdin:
    m = re.match(r'^(\d+):\s*(#+)\s*(.+)', line.strip())
    if m:
        results.append({'line': int(m.group(1)), 'level': len(m.group(2)), 'heading': m.group(3).strip()})
json.dump(results, sys.stdout, ensure_ascii=False, indent=2)
" > harness/robot/night-build/output/YYYY-MM-DD/T-01-E01/headings.json
```

注意：把 YYYY-MM-DD 替换成今天日期。

执行完后，读取输出文件，报告总共找到了多少个标题。
