# 竞品分析：宇树科技 Unitree H1 人形机器人

**调研日期**: 2026-04-02  
**任务编号**: T-0146

---

## 1. 产品概述

宇树科技（Unitree Robotics）于**2023年8月**正式发布首款通用人形机器人H1，被誉为**国内第一台能跑的全尺寸通用人形机器人**。该产品定位于科研和工业应用市场。

---

## 2. 规格参数

| 参数 | 数值 |
|------|------|
| **身高** | 约180cm (1520mm) |
| **体重** | 约47kg |
| **自由度** | 19-27 DOF |
| **单腿自由度** | 髋关节×3 + 膝关节×1 + 踝关节×1 = 5 |
| **单手臂自由度** | 肩关节×3 + 肘关节×1 + 腕关节×3 = 7 |

### 关节扭矩

| 关节 | 极限扭矩 |
|------|----------|
| 膝关节 | 约360 N·m |
| 髋关节 | 约220 N·m |
| 踝关节 | 约45 N·m |
| 手臂关节 | 约75 N·m |

### 运动能力

| 指标 | 数值 |
|------|------|
| **行走速度** | 1.5 m/s |
| **奔跑速度** | 3.3 m/s（世界纪录）|
| **潜在运动性能** | >5 m/s |

### 感知与算力

- **感知系统**: 360°全景深度感知，配备3D激光雷达 + 深度相机
- **电池**: 容量15Ah (864Wh)，最大电压67.2V，支持快速更换
- **算力**: 标配Intel Core i5/i7，选配Nvidia Jetson Orin NX

## 4. 核心功能

### 运动能力
- ✅ 全尺寸双足行走与奔跑
- ✅ 3.3m/s世界纪录奔跑速度
- ✅ 复杂地形自主导航（3D激光SLAM）
- ✅ 高扭矩关节电机（M107），可原地后空翻

### 感知能力
- ✅ 360°全景深度感知
- ✅ 3D激光雷达 + 深度相机
- ✅ 高精度空间数据获取

### 应用场景
- 🏭 工业智造（蔚来汽车工厂搬运任务）
- 🏙️ 城市运营管理
- 🎓 教育科研
- 💪 重体力劳动替代

### 娱乐/展示
- 🎭 2025春晚张艺谋《秧BOT》舞蹈表演
- 🏆 2025世界人形机器人运动会1500米、400米冠军
- 🚗 成功拉动1.4吨汽车负重测试

### 软件生态
- ✅ 人形机器人App Store（2025年12月上线）
- ✅ 预设动作库（扭扭舞、李小龙武术等）
- ✅ 动作捕捉数据移植
- ✅ 动力学算法支持

## 6. 竞争优势

1. **性价比突出**: H1售价低于$89,000，远低于波士顿动力Atlas（未知天价）和特斯拉Optimus
2. **自研核心部件**: 电机、减速器、控制器、激光雷达等核心零部件国产化率超90%
3. **运动性能领先**: 3.3m/s奔跑速度世界纪录
4. **快速迭代**: 依托四足机器人技术积累（G1机器狗全球市占率70%+）

## 7. 市场定位

- **不面向**: 个人消费者（已下架零售渠道）
- **竞品对标**: 波士顿动力Atlas、特斯拉Optimus、Figure 01
