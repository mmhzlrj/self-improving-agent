## 核心要素提炼

### 平台名称
**0-1**（直接叫这个名字）

### 目标
1. 分享打造0-1（具身智能机器人）的过程
2. 吸引对打造自己用的具身智能机器人的人一起玩
3. 内容积累：贵庚的原始记忆、技术经验、动手能力
4. 不圈粉丝，精准定位一部分人

### 内容类型
最起码 **3种内容** 需要AI生成，适合发社媒：
1. 图文内容
2. 视频内容（数字人出镜）
3. （待补充第三种）

### 平台选择
- **优先小红书**：27号禁言解除后考虑，不友好则放弃
- **其他平台均可考虑**：抖音/B站/微博/视频号/YouTube
- **海外平台**：多语种发布

### 数字人出镜
- 用视频大模型生成（SeedDance等）
- 前期需要调研怎么通过数字人形式生成内容

### 标签/定位
- 平台比你更懂算法，先发起来标签自然形成
- 不主动想辐射哪类群众，精准定位一部分人

### 贾维斯概念
- 需要考虑是否融入"贾维斯"概念
- 普通人容易把这个项目和钢铁侠的贾维斯对比

## 建议A序列任务

### 任务名称
**A-影响力平台 0-1 内容生成系统**

### 任务描述
基于OpenClaw，打造0-1影响力平台，内容包括：
1. 数字人视频生成（SVD/SeedDance/海螺等）
2. 多平台多语种内容发布自动化
3. 贵庚记忆积累 → 社媒内容转化
4. 三种社媒内容模板生成

### 阶段规划
- **Phase 1**：数字人视频生成调研 + SeedDanceDemo
- **Phase 2**：多平台发布接口（抖音/B站/YouTube/小红书）
- **Phase 3**：贵庚记忆 → 内容生成自动化 pipeline
- **Phase 4**：多语种翻译 + 海外平台发布
