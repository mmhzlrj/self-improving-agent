## 1. 概述

iPhone Pro 系列（iPhone 12 Pro 起）配备的 LiDAR 扫描仪是获取深度数据的硬件基础。Apple 通过 **ARKit** 和 **AVFoundation** 两套框架向开发者暴露深度数据接口。

**核心数据流：**

```
LiDAR 传感器 (硬件)
    ↓
ARKit SceneDepth API (软件抽象)
    ↓
CVPixelBuffer (Float32 深度图 + 置信度图)
    ↓
OpenClaw iOS Bridge (传输层)
    ↓
主机端 (处理/可视化)
```

## 2. ARKit Depth API 使用方法

### 2.1 启用方式

通过 `ARWorldTrackingConfiguration` 启用，配置 `frameSemantics` 为 `.sceneDepth`：

```swift
import ARKit

class ARDepthViewController: UIViewController, ARSessionDelegate {

    var session: ARSession!

    func startAR() {
        guard ARWorldTrackingConfiguration.isSupported else { return }

        let configuration = ARWorldTrackingConfiguration()
        configuration.frameSemantics = [.sceneDepth]  // 启用深度感知

        session = ARSession()
        session.delegate = self
        session.run(configuration)
    }

    // MARK: - ARSessionDelegate

    func session(_ session: ARSession, didUpdate frame: ARFrame) {
        // 方式1：通过 sceneDepth 获取（推荐，ARKit 4+）
        guard let sceneDepth = frame.sceneDepth else { return }
        let depthMap = sceneDepth.depthMap        // CVPixelBuffer
        let confidenceMap = sceneDepth.confidenceMap // CVPixelBuffer

        // 方式2：通过 rawDepth 获取（LiDAR 原始数据）
        if let rawDepth = frame.rawDepth {
            let rawDepthMap = rawDepth.depthMap
            let rawConfidenceMap = rawDepth.confidenceMap
        }
    }
}
```

### 2.2 ARFrame 深度数据接口

| 属性 | 类型 | 说明 |
|------|------|------|
| `sceneDepth` | `ARDepthData?` | 融合后的深度图（LiDAR + 摄像头融合，质量更高） |
| `rawDepth` | `ARDepthData?` | LiDAR 原始稀疏深度数据（ARKit 5+） |

### 2.3 ARDepthData 结构

```swift
@available(iOS 13.4, *)
public struct ARDepthData {
    public var depthMap: CVPixelBuffer  // 深度图
    public var confidenceMap: CVPixelBuffer  // 置信度图
}
```

## 3. 深度数据格式详解

### 3.1 深度图（depthMap）

| 属性 | 值 |
|------|------|
| **Pixel Format** | `kCVPixelFormatType_32Float` (Float32, 32位单精度浮点) |
| **单位** | 米（meters） |
| **分辨率** | **256 × 192**（原生 LiDAR 融合后） |
| **每像素含义** | 该像素对应现实世界中某点的深度值 |

**如何读取 Float32 深度值：**

```swift
func readDepthValues(from depthMap: CVPixelBuffer) -> [(x: Int, y: Int, depth: Float)] {
    let width = CVPixelBufferGetWidth(depthMap)
    let height = CVPixelBufferGetHeight(depthMap)

    CVPixelBufferLockBaseAddress(depthMap, .readOnly)
    defer { CVPixelBufferUnlockBaseAddress(depthMap, .readOnly) }

    let floatBuffer = CVPixelBufferGetBaseAddress(depthMap)!.assumingMemoryBound(to: Float32.self)

    var results: [(x: Int, y: Int, depth: Float)] = []
    results.reserveCapacity(width * height)

    for y in 0..<height {
        for x in 0..<width {
            let depth = floatBuffer[y * width + x]
            if depth > 0 {  // 0 表示无效深度
                results.append((x: x, y: y, depth: depth))
            }
        }
    }
    return results
}
```

### 3.2 置信度图（confidenceMap）

| 属性 | 值 |
|------|------|
| **Pixel Format** | `kCVPixelFormatType_8Confidence` (UInt8, 0-255) |
| **值含义** | 0=低置信度, 1=中, 2=高 |
| **用途** | 过滤低质量深度点 |

```swift
func readConfidence(from confidenceMap: CVPixelBuffer) -> [[UInt8]] {
    let width = CVPixelBufferGetWidth(confidenceMap)
    let height = CVPixelBufferGetHeight(confidenceMap)

    CVPixelBufferLockBaseAddress(confidenceMap, .readOnly)
    defer { CVPixelBufferUnlockBaseAddress(confidenceMap, .readOnly) }

    let uint8Buffer = CVPixelBufferGetBaseAddress(confidenceMap)!.assumingMemoryBound(to: UInt8.self)

    var matrix = [[UInt8]](repeating: [UInt8](repeating: 0, count: width), count: height)
    for y in 0..<height {
        for x in 0..<width {
            matrix[y][x] = uint8Buffer[y * width + x]
        }
    }
    return matrix
}
```

### 3.3 深度数据分辨率说明

> ⚠️ **重要：原生 LiDAR 原始分辨率 ≠ 输出分辨率**

- **LiDAR 物理采样率**：Apple LiDAR 原始输出约为 **9×64** 稀疏点阵
- **ARKit 上采样**：通过与 RGB 摄像头帧 + IMU 惯性数据融合，上采样到 **256×192** 密集深度图
- **对齐 RGB 摄像头**：如果需要与高清摄像头画面叠加，需要使用 **CILanczosScaleTransform** 或类似算法将 256×192 上采样到 RGB 分辨率（如 1920×1440）

**上采样代码参考：**

```swift
import CoreImage

func upscaleDepthToMatchRGB(depthPixelBuffer: CVPixelBuffer, targetSize: CGSize) -> CVPixelBuffer? {
    let ciImage = CIImage(cvPixelBuffer: depthPixelBuffer)

    let scaleFilter = CIFilter(name: "CILanczosScaleTransform")!
    scaleFilter.setValue(ciImage, forKey: kCIInputImageKey)

    let currentWidth = CGFloat(CVPixelBufferGetWidth(depthPixelBuffer))
    let currentHeight = CGFloat(CVPixelBufferGetHeight(depthPixelBuffer))

    let scaleX = targetSize.width / currentWidth
    let scaleY = targetSize.height / currentHeight

    scaleFilter.setValue(scaleX, forKey: kCIInputScaleKey)
    scaleFilter.setValue(scaleY, forKey: kCIInputAspectRatioKey)

    let context = CIContext()
    guard let outputImage = scaleFilter.outputImage else { return nil }

    var outputBuffer: CVPixelBuffer?
    let attrs: [String: Any] = [
        kCVPixelBufferCGImageCompatibilityKey as String: true,
        kCVPixelBufferCGBitmapContextCompatibilityKey as String: true
    ]

    CVPixelBufferCreate(nil, Int(targetSize.width), Int(targetSize.height),
                        kCVPixelFormatType_32Float, attrs as CFDictionary, &outputBuffer)

    if let buffer = outputBuffer {
        context.render(outputImage, to: buffer)
    }
    return outputBuffer
}
```

## 4. 获取频率和分辨率

| 参数 | 值 | 说明 |
|------|------|------|
| **帧率** | **60 fps** | iPhone Pro 机型典型帧率 |
| **深度图分辨率** | **256 × 192** | ARKit 融合后输出 |
| **置信度图分辨率** | **256 × 192** | 与深度图相同 |
| **RGB 摄像头分辨率** | 最高 1920×1440+ | 不同机型支持不同 |
| **深度范围** | **0 ~ 5 米**（官方规格）实际可用到 ~10米（Apple specs） | Apple 官方标称5米 |
| **精度** | 毫米级（< 1% 误差）| 在5米范围内 |

class DepthCaptureService: NSObject {

    }

### 5.2 从深度数据生成 3D 点云

```swift
import simd

struct PointCloudGenerator {

    /// 从深度图和相机内参生成世界坐标点云
    func generatePointCloud(
        depthMap: CVPixelBuffer,
        cameraIntrinsics: simd_float3x3
    ) -> [SIMD3<Float>] {

        let width = CVPixelBufferGetWidth(depthMap)
        let height = CVPixelBufferGetHeight(depthMap)

        CVPixelBufferLockBaseAddress(depthMap, .readOnly)
        defer { CVPixelBufferUnlockBaseAddress(depthMap, .readOnly) }

        let depthPtr = CVPixelBufferGetBaseAddress(depthMap)!.assumingMemoryBound(to: Float32.self)

        var points: [SIMD3<Float>] = []

        for y in 0..<height {
            for x in 0..<width {
                let depth = depthPtr[y * width + x]

                guard depth > 0 else { continue }

                // 将像素坐标 + 深度 → 相机坐标系下的 3D 点
                // fx = intrinsics[0][0], fy = intrinsics[1][1]
                // cx = intrinsics[2][0], cy = intrinsics[2][1]
                let fx = cameraIntrinsics[0][0]
                let fy = cameraIntrinsics[1][1]
                let cx = cameraIntrinsics[2][0]
                let cy = cameraIntrinsics[2][1]

                let worldX = (Float(x) - cx) * depth / fx
                let worldY = (Float(y) - cy) * depth / fy
                let worldZ = depth

                points.append(SIMD3<Float>(worldX, worldY, worldZ))
            }
        }
        return points
    }

    /// 获取 ARFrame 的相机内参
    func getCameraIntrinsics(from frame: ARFrame) -> simd_float3x3 {
        return frame.camera.intrinsics  // 直接从 ARFrame 获取
    }
}
```

### 6.2 iOS 端实现

**深度数据捕获模块：**

```swift
import Foundation
import ARKit
import Compression

class LiDARBridgeService: NSObject {

    private var session: ARSession?
    private var webSocketTask: URLSessionWebSocketTask?

    var openClawHostURL: String = "ws://<OPENCLAW_HOST>:18888/lidar"

    // MARK: - 公开 API

    func connect() {
        // 1. 启动 ARKit
        startARSession()

        // 2. 连接 OpenClaw WebSocket
        connectWebSocket()
    }

    func disconnect() {
        session?.pause()
        webSocketTask?.cancel(with: .goingAway, reason: nil)
    }

    // MARK: - 私有

    private func startARSession() {
        let config = ARWorldTrackingConfiguration()
        config.frameSemantics = [.sceneDepth]
        config.videoResolution = .hd1920x1080

        session = ARSession()
        session?.delegate = self
        session?.run(config)
    }

    private func connectWebSocket() {
        guard let url = URL(string: openClawHostURL) else { return }

        let ws = URLSession.webSocketTask(with: url)
        ws?.resume()
        webSocketTask = ws

        // 监听消息
        receiveMessage()
    }

    private func receiveMessage() {
        webSocketTask?.receive { [weak self] result in
            switch result {
            case .success(let message):
                if case .string(let text) = message {
                    self?.handleCommand(text)
                }
                self?.receiveMessage()
            case .failure(let error):
                print("WebSocket 错误: \(error)")
            }
        }
    }

    private func handleCommand(_ command: String) {
        switch command {
        case "start": startARSession()
        case "stop":  session?.pause()
        case "subscribe":
            // 开始持续发送深度数据
            isStreaming = true
        case "unsubscribe":
            isStreaming = false
        default:
            break
        }
    }

    private var isStreaming = false

    private func sendDepthData(_ depthMap: CVPixelBuffer, _ confidenceMap: CVPixelBuffer) {
        guard isStreaming else { return }

        // 1. 读取 Float32 深度数据
        let width = CVPixelBufferGetWidth(depthMap)
        let height = CVPixelBufferGetHeight(depthMap)

        CVPixelBufferLockBaseAddress(depthMap, .readOnly)
        defer { CVPixelBufferUnlockBaseAddress(depthMap, .readOnly) }

        let floatPtr = CVPixelBufferGetBaseAddress(depthMap)!
            .assumingMemoryBound(to: Float32.self)

        // 2. 转换为字节数组
        let byteCount = width * height * MemoryLayout<Float32>.size
        let bytePtr = floatPtr.withMemoryRebound(to: UInt8.self, capacity: byteCount) { $0 }
        let depthBytes = Data(bytes: bytePtr, count: byteCount)

        // 3. 构建 JSON 元数据 + 压缩
        let metadata: [String: Any] = [
            "type": "lidar_depth",
            "width": width,
            "height": height,
            "timestamp": Date().timeIntervalSince1970
        ]

        // 4. 压缩传输（zstd 或 lz4，比 gzip 快）
        if let compressed = compressDepthData(depthBytes) {
            sendBinaryFrame(compressed, metadata: metadata)
        }
    }

    private func compressDepthData(_ data: Data) -> Data? {
        // 使用 zstd 压缩，压缩比高且速度快
        let destinationBuffer = UnsafeMutablePointer<UInt8>.allocate(capacity: data.count)
        defer { destinationBuffer.deallocate() }

        let compressedSize = data.withUnsafeBytes { srcPtr in
            compression_encode_buffer(
                destinationBuffer, data.count,
                srcPtr.bindMemory(to: UInt8.self).baseAddress!, data.count,
                nil, COMPRESSION_ZSTD)
        }

        guard compressedSize > 0 else { return nil }
        return Data(bytes: destinationBuffer, count: compressedSize)
    }

    private func sendBinaryFrame(_ data: Data, metadata: [String: Any]) {
        guard let jsonData = try? JSONSerialization.data(withJSONObject: metadata),
              let jsonString = String(data: jsonData, encoding: .utf8) else { return }

        let message = URLSessionWebSocketTask.Message.string(jsonString)
        webSocketTask?.send(message) { error in
            if let error = error { print("发送失败: \(error)") }
        }

        // 发送二进制数据
        let binaryMessage = URLSessionWebSocketTask.Message.data(data)
        webSocketTask?.send(binaryMessage) { _ in }
    }
}

// MARK: - ARSessionDelegate
extension LiDARBridgeService: ARSessionDelegate {

    func session(_ session: ARSession, didUpdate frame: ARFrame) {
        guard let sceneDepth = frame.sceneDepth, isStreaming else { return }
        sendDepthData(sceneDepth.depthMap, sceneDepth.confidenceMap)
    }
}
```

### 6.3 主机端（OpenClaw）接收处理

**Python 接收端示例（harness/robot 模块）：**

```python
import asyncio
import json
import zstandard as zstd
import numpy as np
from dataclasses import dataclass
from typing import Optional

@dataclass
class LiDARFrame:
    width: int
    height: int
    timestamp: float
    depth_data: np.ndarray  # shape: (height, width), dtype: float32

class LiDARReceiver:
    """接收 iOS LiDAR 深度数据并转换为点云/障碍物信息"""

    def __init__(self, host: str = "0.0.0.0", port: int = 18888):
        self.host = host
        self.port = port
        self.latest_frame: Optional[LiDARFrame] = None

    async def start(self):
        server = await asyncio.start_server(
            self._handle_client, self.host, self.port
        )
        print(f"[LiDAR] 监听 {self.host}:{self.port}")
        async with server:
            await server.serve_forever()

    async def _handle_client(self, reader: asyncio.StreamReader, writer: asyncio.StreamWriter):
        buffer = b""
        metadata: Optional[dict] = None

        while True:
            data = await reader.read(8192)
            if not data:
                break

            buffer += data

            # 尝试解析 JSON 元数据
            if metadata is None:
                try:
                    # 查找 JSON 边界（换行符分隔）
                    lines = buffer.split(b'\n')
                    for line in lines:
                        if line.strip():
                            metadata = json.loads(line.decode())
                            buffer = buffer[len(line)+1:]
                            break
                    continue
                except (json.JSONDecodeError, UnicodeDecodeError):
                    continue

            # 解压缩深度数据
            if metadata and len(buffer) >= 1024:
                depth_bytes = self._decompress(buffer)
                if depth_bytes:
                    self._process_depth(depth_bytes, metadata)
                    buffer = b""
                    metadata = None

    def _decompress(self, data: bytes) -> Optional[bytes]:
        try:
            dctx = zstd.ZstdDecompressor()
            return dctx.decompress(data)
        except Exception as e:
            print(f"[LiDAR] 解压失败: {e}")
            return None

    def _process_depth(self, depth_bytes: bytes, metadata: dict):
        width = metadata["width"]
        height = metadata["height"]

        depth_array = np.frombuffer(depth_bytes, dtype=np.float32)
        depth_array = depth_array.reshape((height, width))

        self.latest_frame = LiDARFrame(
            width=width,
            height=height,
            timestamp=metadata["timestamp"],
            depth_data=depth_array
        )

        # 生成障碍物检测结果
        self._detect_obstacles(depth_array)

    def _detect_obstacles(self, depth_array: np.ndarray, threshold_m: float = 1.5):
        """检测近距离障碍物（深度 < threshold_m 的区域）"""
        mask = depth_array < threshold_m
        if mask.any():
            # 计算障碍物区域边界
            rows, cols = np.where(mask)
            print(f"[LiDAR] 🚧 障碍物检测: "
                  f"范围 行{rows.min()}-{rows.max()}, "
                  f"列{cols.min()}-{cols.max()}, "
                  f"距离 {depth_array[mask].min():.2f}m - {depth_array[mask].max():.2f}m")
```

### 6.4 数据传输格式

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | string | `"lidar_depth"` |
| `width` | int | 256 |
| `height` | int | 192 |
| `timestamp` | float | Unix 时间戳（秒） |
| `compressed_data` | binary | Zstd 压缩后的 Float32 原始字节 |

**原始帧大小（未压缩）：**
```
256 × 192 × 4 bytes = 196,608 bytes ≈ 192 KB
```

**压缩后（Zstd）：**
```
≈ 20-50 KB（取决于场景复杂度）
```

**带宽估算（60fps）：**
```
≈ 1.2 - 3 MB/s（压缩后）
```

## 8. 已知限制与注意事项

1. **热限制**：长时间使用 LiDAR 会导致设备发热降频，帧率可能从 60fps 降到 30fps
2. **距离限制**：有效测距 ~5 米，超过后深度值可靠性下降
3. **透明/反光物体**：深度数据可能出现空洞或错误值
4. **阳光直射**：强光环境下深度精度降低
5. **数据量**：每帧 192KB（未压缩），高频传输需要压缩或降低帧率
6. **沙盒限制**：iOS 应用无法直接访问其他应用的数据，需要独立 ARKit Session
