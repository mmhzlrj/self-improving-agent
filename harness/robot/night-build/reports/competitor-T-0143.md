## 三、技术架构

### 3.1 计算平台

| 组件 | 配置 |
|:---|:---|
| **主控芯片** | **NVIDIA Jetson Xavier NX** |
| **AI算力** | **21 TOPS**（INT8） |
| **CPU** | 6核 Carmel ARM v8.2 64位处理器 |
| **GPU** | 384核 Volta架构，48个 Tensor Core |
| **内存** | 8GB LPDDR4x |
| **存储** | 16GB eMMC + 32GB SD卡 |
| **协处理器** | 运动控制（MR813）+ 语音交互（R329） |

### 3.2 运动执行层

- **自研 CyberGear 微伺服电机**：共 **12 个**（每腿 3 个）
- **单个电机参数**：
  - 重量：317g
  - 最大扭矩：12 N·m
  - 最大转速：220 rpm
  - 控制频率：1000Hz
- **整机峰值扭矩密度**：37.85 N·m/kg

### 3.3 软件系统

- Ubuntu + ROS 2 机器人操作系统
- 开放 **100+ 功能接口**
- 公开视觉算法和结构图纸
- 支持 OTA 升级

## 五、运动性能

| 参数 | 具体数据 |
|:---|:---|
| **最大行走速度** | **1.6 m/s**（约 5.76 km/h） |
| **最大奔跑速度** | **3.2 m/s**（约 11.5 km/h） |

| **最大越障高度** | **10 cm** |
| **最大负载** | **1 kg** |

### 5.1 核心能力

- 动态平衡算法
- 摔倒保护、倒地自恢复
- 支持步态：慢走、快走、小跑、变频、四足跳跑

### 5.2 特技动作

站立、趴下、坐下、作揖、俯卧撑、太空步、芭蕾舞、原地跳跃、前后空翻（实验版）、滑滑板（实验版）

### 5.3 精细控制

- "前足踩豆腐，豆腐不破" 展示精细力控
- 被外力推搡后迅速恢复平衡

## 六、与 0-1 项目的差异化分析

| 维度 | 小米 CyberDog 2 | 0-1 项目（推测） |
|:---|:---|:---|
| **价格** | 12,999 元 | 待定（预计更低） |
| **算力** | 21 TOPS（Jetson NX） | 预计更低（低成本方案） |
| **传感器数量** | 19 个（高端配置） | 预计更少（精简方案） |
| **自由度** | 12 个 | 相近 |
| **电机** | 自研 CyberGear（12个） | 待确认 |
| **目标用户** | 开发者/极客 | 待确认 |
| **系统开放性** | ROS 2 + 开源 SDK | 待确认 |
| **AI 能力** | 小爱同学 + 大模型 | 待确认 |

### 6.1 差异化机会

1. **成本优势**：CyberDog 2 定价 1.3 万，0-1 可考虑更低价格区间
2. **精简传感器方案**：去除激光雷达等高成本传感器，采用纯视觉方案
3. **轻量化部署**：使用更低算力芯片（如 Jetson Orin Nano 或入门级方案）
4. **应用场景聚焦**：专注特定场景（如教育陪伴 vs 开发者平台）
