### ⚠️ ollama.ai vs ollama.com

**映射报告声明**: `curl -fsSL https://ollama.ai/install.sh | sh`
**验证结论**: **错误**（GLM 指出）

Ollama 官方下载地址是 **ollama.com**，不是 ollama.ai。ollama.ai 可能被占用或重定向，官方脚本应使用 ollama.com。

**修正建议**:
```bash
# 原文（错误）
curl -fsSL https://ollama.ai/install.sh | sh
# 修正为
curl -fsSL https://ollama.com/install.sh | sh
```
