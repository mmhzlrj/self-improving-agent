## 1. 2D 游戏化工作空间的视觉设计

### 色彩体系

Homescapes/Gardenscapes 均由 Playrix 开发，采用温暖、治愈系的色彩体系：

- **主色调**：柔和的大地色系（暖棕、米白、淡黄）作为庄园/花园场景基调
- **功能色**：高饱和度的暖色（明黄、橙红、翠绿）用于按钮、金币、星星等 UI 元素
- **背景与 UI 对比**：场景采用柔和渐变 + 手绘纹理；UI 元素使用半透明面板或木质/金属边框，与场景形成区分
- **情绪色**：游戏化反馈大量使用金色（奖励）、绿色（成功/完成）、红色（限时/紧迫感）

> ⚠️ 以上色值未找到官方色卡文档，基于截图观察估算。

### 图标风格

- **超写实 + 扁平混合**：图标保留真实材质感（木材纹理、金属光泽），同时加入扁平化的轮廓线条和微渐变
- **功能拟物化**：木质家具 = 装修功能，星形 = 进度/等级，钱袋 = 虚拟货币
- **内阴影 + 高光**：图标普遍带有内阴影增加立体感，顶部高光模拟光照质感
- **圆角矩形**：大多数图标容器使用圆角矩形边框，视觉上柔和友好

### 界面布局

- **场景嵌入式**：游戏主界面本身就是一幅手绘风格的家宅/花园场景，UI 元素悬浮于场景之上（而非独立面板）
- **顶部状态栏**：金币数量、星星数量、绿钞（稀有货币）、体力值，位于屏幕顶部
- **底部工具栏**：主要操作入口（地图、任务、商店、社交），图标尺寸较大，便于触控
- **浮动装饰**：装修进度条、每日任务通知以半透明卡片形式出现在场景中，不遮挡核心画面

## 2. 进度系统设计

### 等级与关卡

- **线性关卡地图**：以室内/花园为单位的横版地图，每个关卡用星星表示，玩家消耗体力通关
- **剧情驱动的装修**：通关获得的星星用于"修复"庄园中的具体物品（喷泉、墙壁、家具），before/after 视觉对比强烈
- **章节解锁**：地图分段解锁（如"主宅""侧楼""花园"），形成长线目标

### 奖励机制

- **多层递进奖励**：通关奖励、转盘、每日登录、限时活动多轨并行
- **开箱/翻牌动画**：奖励发放界面设计为礼物盒开启、翻牌、转盘等，强调惊喜感和仪式感
- **活动赛季通行证**：特定活动有独立的赛季进度条，奖励更稀有（独家装饰、稀有货币）
- **连续登录**：以"日历"形式展示，每天一个礼物，激励每日登录

### 每日任务

- **剧情包装**：任务以角色对话形式呈现（如"奥斯汀需要你帮忙清理喷泉"），而非生硬的"完成任务 X"
- **日志本/日历隐喻**：任务列表用笔记本或日志本的视觉载体，降低"工作感"
- **动态难度**：每日任务难度与玩家等级挂钩，保证成就感

## 3. 核心交互模式

### 虚拟货币

| 货币 | 用途 | 视觉呈现 |
|------|------|----------|
| 金币 | 购买道具、体力补充 | 金色硬币堆，带高光 |
| 星星 | 核心装修资源 | 黄色星形，带飞行粒子特效 |
| 绿钞/红宝石 | 稀有装饰、秒刷体力 | 绿色钞票/宝石，极少获得 |

- **粒子飞入**：星星/金币获得时，以抛物线轨迹飞入顶部状态栏，强调收集感
- **溢出动画**：获得大量货币时，有数字滚动 + 光效叠加

### 装饰系统

- **Before & After 对比**：点击可装修对象时，弹出选择面板，展示修复前（布满灰尘/损坏）vs 修复后（新装/干净）的对比截图
- **轮盘/列表选择**：提供多个装修风格选项，视觉预览嵌入面板中
- **渐进解锁**：随着装修进度推进，场景视觉持续发生变化，给玩家"亲手改造"的成就感

### 社交系统

- **轻量化社交**：体力赠送（点击即送）、排行榜（仅排名，无深度交互）
- **公会/小队**：玩家可加入公会，但 UI 上仅提供简单的聊天和任务共享
- **独立社交入口**：通常在屏幕左侧或底部有单独的"社交建筑"图标，不干扰主界面

## 4. 动画和动效特点

- **Squash & Stretch**：按钮点击有轻微的按压缩回弹效果
- **粒子特效**：星星飞入、金币获得时有粒子拖尾
- **场景特写**：装修完成时，物体会播放一段"组装"或"清洁"动画（而非简单的渐显消失）
- **镜头转场**：关卡界面 → 庄园主界面采用平滑推拉（Zoom）或滑动（Slide），保持空间连续性
- **弹性入场**：UI 面板弹出时带有轻微弹性（spring animation），视觉上更生动
- **待机循环**：场景中的装饰物（如喷泉流水、风车旋转）有微循环待机动画，保持画面活力

## 5. 对 0-1 项目 3D 世界的借鉴建议

### 设计语言选择

- **场景嵌入式 UI**：将 UI 融入 3D 场景，而非独立面板；考虑使用半透明玻璃、金属边框等材质让 UI"悬浮"在世界中
- **温暖治愈系配色**：以柔和的大地色为基底，高饱和暖色为点缀，营造"家"的氛围
- **Before & After 动态**：关键交互（如建造、升级）加入灰尘消散/组装完成的特写动画，强化正反馈

### 进度系统建议

- **剧情驱动进度**：任务包装成角色请求而非机械清单，降低"肝感"
- **多层奖励叠加**：体力/金币/稀有货币三轨并行，配合每日/每周/赛季的多层奖励
- **日历化登录**：用日历 UI 承载每日奖励，制造连续登录动机

### 交互反馈建议

- **粒子飞入**：收集类资源使用抛物线粒子动画（推荐用 Canvas/WebGL 实现）
- **弹性动效**：UI 弹出/切换使用 spring animation，避免生硬的 fade
- **待机动画**：场景装饰物加入微循环动画，保持画面活力
