# Polymarket Smart Money AI Agent

> 识别盈利交易员，生成跟单信号

## 功能

- 📊 **链上数据分析**：抓取 Polymarket 交易数据
- 🧠 **智能筛选**：识别持续盈利的交易员
- 📈 **模式分析**：分析交易员的下注模式
- 📡 **信号推送**：将优质信号推送到执行机器人

## 核心指标

| 指标 | 说明 |
|------|------|
| **ROI** | 投资回报率 |
| **胜率** | 盈利交易占比 |
| **盈亏比** | 平均盈利/平均亏损 |
| **交易频率** | 活跃度 |
| **持仓周期** | 平均持仓时间 |

## 命令

| 命令 | 说明 |
|------|------|
| `查看热门交易员` | 显示 Top 10 盈利交易员 |
| `分析 交易员地址` | 分析特定交易员表现 |
| `监听交易员` | 开始监控特定交易员 |
| `设置策略` | 配置筛选条件 |
| `推送信号` | 手动触发信号推送 |

## 数据来源

- Polymarket API (polymarket.com)
- Polygon 链上数据
- Dune Analytics (可选)

## 注意事项

⚠️ 本模块仅分析数据，不执行交易
⚠️ 历史表现不代表未来收益
⚠️ 请遵守当地法律法规
