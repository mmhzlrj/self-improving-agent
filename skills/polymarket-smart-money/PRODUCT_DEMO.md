# 🤖 Polymarket Smart Money AI Agent

## 产品愿景

**帮助用户识别最赚钱的交易员，并自动复制他们的交易策略。**

---

## 核心功能

| 功能 | 描述 |
|------|------|
| 🏆 **顶级交易员识别** | 从链上数据中识别持续盈利的交易员 |
| 📊 **交易模式分析** | 分析交易员的风格、擅长领域、风险偏好 |
| 📡 **实时信号推送** | 当顶级交易员新开仓位时推送信号 |
| 💰 **智能跟单** | 按比例复制交易（他买1000，你买10-100） |
| 📈 **批量跟单** | 同时复制多个交易员，分散风险 |

## 如何工作

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  Polymarket    │───►│   AI 分析引擎   │───►│   跟单信号      │
│  链上数据       │    │  识别盈利交易员  │    │  推送给用户     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                              │
                              ▼
                       ┌─────────────────┐
                       │  执行机器人     │
                       │  自动复制交易   │ (可选)
                       └─────────────────┘
```

---

## 技术实现

1. **数据层**: Polymarket API + Polygon 链上数据
2. **分析层**: LLM AI 分析交易模式 + 策略识别
3. **信号层**: 实时推送 + 过滤噪音
4. **执行层**: 对接已有交易机器人（可选）

---

## 产品优势

- ⚡ **远超人工**: 能处理数千个地址的交易数据
- 🧠 **AI 驱动**: LLM 分析交易模式，发现人类难以察觉的规律
- 🔄 **24/7 监控**: 不间断监控顶级交易员动态
- 🎯 **精准信号**: 只推送高置信度信号，减少噪音
- 🔒 **可控**: 用户设置跟单比例，完全控制风险

---

## 限制说明

- 历史表现不代表未来收益
- 无法完全消除链上滑点/延迟
- 无法检测隐蔽的协调恶意交易
- 性能取决于集成的 LLM 能力
