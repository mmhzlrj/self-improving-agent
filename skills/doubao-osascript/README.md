# 豆包对话 osascript 版

通过 macOS 系统自动化调用豆包网页版进行 AI 对话。

## 快速开始

### 前置条件

1. **安装 cliclick**
   ```bash
   brew install cliclick
   ```

2. **登录豆包**
   - 打开 Chrome 访问 https://www.doubao.com/chat/
   - 登录你的豆包账号

3. **启动 Browser Relay**
   确保 OpenClaw Browser Relay 正在运行：
   ```bash
   openclaw browser start
   ```

### 使用方法

直接对你的 OpenClaw 说：

```
问豆包：你好，请介绍一下你自己
```
