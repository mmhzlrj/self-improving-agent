# 🦞 小龙虾 P2P 聊天 Skill 使用说明

> 不需要服务器的直接聊天工具

---

## 这是什么？

小龙虾 P2P 是一种**去中心化**的聊天方式——两台电脑直接连接发消息，不经过任何服务器。

**适用场景：**
- 安装工直接联系业主（不经过平台）
- 用户之间私密沟通
- 不想被平台监控/抽成

---

## 安装步骤

### 第1步：获取安装包

把 `lobster-p2p.zip` 文件放到这个目录：
```
~/.openclaw/workspace/skills/
```

### 第2步：解压

```bash
cd ~/.openclaw/workspace/skills/
unzip lobster-p2p.zip -d lobster-p2p
```

### 第3步：重启 OpenClaw

```bash
# 停止
Ctrl+C 关闭当前运行的 OpenClaw

# 重新启动
openclaw gateway
```

---

## 快速开始

### 1. 启动 P2P 服务器

告诉小龙虾：
```
启动 P2P
```

### 2. 告诉对方你的地址

告诉小龙虾：
```
我的 P2P 信息
```

它会显示类似这样的信息：
```
📡 你的 P2P 信息：
🔹 昵称: 小龙虾-xxx
🔹 端口: 5566
🔹 设备ID: xxxxxxxxxx
```

**把 "IP:端口" 发给对方**（微信/飞书都可以）

### 3. 连接对方并发送消息

对方告诉你他的地址后，执行：
```
连接 192.168.x.x:5566 你好！
```

---

## 常用命令

| 命令 | 作用 |
|------|------|
| `启动 P2P` | 开启 P2P 服务器（只需一次） |
| `我的 P2P 信息` | 查看你的地址 |
| `连接 IP:端口 消息` | 连接对方并发送消息 |
| `联系人列表` | 查看已加过的好友 |
| `我的昵称 张三` | 设置你的昵称 |
| `我的标签 安装工` | 设置你的标签 |

---

## 注意事项

⚠️ **网络要求：**
- 目前需要在**同一 WiFi/局域网**才能连接
- 双方都要运行"启动 P2P"

⚠️ **地址会变：**
- 如果 IP 地址变了，需要重新告诉对方新地址
