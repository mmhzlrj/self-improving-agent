# 吸盘抓手模块

> 气泵 + 电磁阀控制，抓取平面物体，由机械臂模块统一调度

## 概述

吸盘抓手（Suction Gripper）是 0-1 机器人执行物理抓取任务的末端执行器之一。与传统机械爪不同，吸盘抓手利用负压吸附原理，适用于抓取平面、光滑的物体（如金属板、玻璃、塑料片等）。气泵抽气形成真空负压，电磁阀控制吸附/释放时序。当前状态为待定（方案设计阶段），与机械臂 Phase 4 协同推进。

## 硬件参数

| 参数 | 规格 |
|------|------|
| 吸盘类型 | 硅胶真空吸盘（平面/凹槽型）|
| 气泵 | 12V微型真空泵（最大真空度 -90kPa）|
| 电磁阀 | 12V 二通电磁阀（常闭型，通电打开）|
| 控制电压 | ESP32 GPIO 3.3V → 继电器模块 → 12V 控制 |
| 通信协议 | 由机械臂 Cyber Bricks 统一通过 MQTT 指令调度 |
| 适用物体 | 平面、光滑、厚度 >2mm 的硬质材料 |

## 技术方案

**控制逻辑**：
```
机械臂 Cyber Bricks（统一调度）
    ↓ MQTT 指令
电磁阀控制引脚（GPIO → 继电器）
    ↓
气泵供电通断（12V）
    ↓
吸盘吸附/释放
```

**时序控制**：
| 步骤 | 动作 | 时长 |
|------|------|------|
| 1 | 电磁阀打开（放气，吸盘脱离）| 0.2s |
| 2 | 吸盘压合到物体表面 | 0.3s |
| 3 | 电磁阀关闭（气泵抽气，负压吸附）| 0.5s |
| 4 | 机械臂执行搬运动作 | — |
| 5 | 到达目标位置，电磁阀打开（放气解除吸附）| 0.2s |

**硬件接线**：
| 连接 | 说明 |
|------|------|
| ESP32 GPIO → 继电器模块 IN | 3.3V 信号控制 |
| 继电器模块 COM → 12V 气泵正极 | 电源开关 |
| 12V 气泵负极 → GND | 电源回路 |
| 电磁阀 → 与气泵并联或单独控制 | 可选：独立控制吸盘开合 |

**MQTT 指令格式**（由机械臂模块统一发送）：
```json
// 吸附命令
{"type": "suction", "action": "attach"}

// 释放命令
{"type": "suction", "action": "detach"}

// 查询状态
{"type": "suction", "action": "status"}
```

## 问题记录

**漏气导致吸附力不足**
- **现象**：吸盘无法有效抓取物体，提前脱落
- **原因**：吸盘边缘有灰尘/杂物，或物体表面不平整
- **解决方案**：
  1. 每次抓取前清洁吸盘边缘和物体表面
  2. 选用硬度更高的硅胶吸盘（耐磨性更好）
  3. 增加负压检测：接压力传感器，负压 <阈值时判定为漏气

**气泵噪音**
- **现象**：12V 真空泵工作噪音大（可达 60-70dB）
- **原因**：微型气泵机械结构决定
- **解决方案**：
  1. 使用隔音箱或软胶减震垫
  2. 改用微型无刷真空泵（成本更高但更安静）
  3. 吸附完成后立即关闭气泵（保持负压不持续运行，减少噪音）

**吸取重量受限**
- **现象**：吸盘无法抓取重物
- **原因**：负压吸附力与吸盘面积、真空度成正比
- **解决方案**：
  1. 根据物体重量选配更大直径吸盘
  2. 多吸盘并联（多个小吸盘同时吸附，分担重量）
  3. 重物（>1kg）建议改用机械爪方案
