# 智库脚本测试用例

## 测试目标
验证"一键群发 + 自动获取回复"功能

## 前置条件
1. Gateway 运行中
2. Browser Relay 已连接
3. 5 个平台已登录

## 测试用例

### 测试1: 简单问题
```bash
~/.openclaw/workspace/scripts/zhiku.sh "1+1等于几"
```
**预期**：快速得到回复
**验证点**：
- [ ] 发送到 5 个平台
- [ ] 获取到 5 个平台的回复

### 测试2: 联网问题
```bash
~/.openclaw/workspace/scripts/zhiku.sh "今天深圳天气怎么样"
```
**预期**：需要联网搜索
**验证点**：
- [ ] 发送到 5 个平台
- [ ] 智谱/千问显示联网搜索结果

### 测试3: 开放问题
```bash
~/.openclaw/workspace/scripts/zhiku.sh "用一句话介绍深圳"
```
**预期**：各平台风格不同
**验证点**：
- [ ] 发送到 5 个平台
- [ ] 各平台回复内容不同
