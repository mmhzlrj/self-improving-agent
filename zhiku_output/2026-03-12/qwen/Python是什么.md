Python 是一种高级、解释型、通用的编程语言。
简单来说，它是一种让计算机执行任务的工具，但它的语法非常接近人类语言（英语），因此以易读、易写、易学而闻名。
以下是关于 Python 的核心介绍：
1. 核心特点
语法简洁： Python 强调代码的可读性，使用缩进（空格）来划分代码块，而不是大括号。这使得代码看起来非常整洁。
解释型语言： 代码不需要编译成二进制文件，而是由解释器逐行运行，方便调试和测试。
跨平台： 可以在 Windows、macOS、Linux 等多种操作系统上运行。
丰富的库： 拥有庞大的标准库和第三方库生态系统（被称为“电池Included"），你可以找到几乎任何功能的现成代码。
