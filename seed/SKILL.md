# Seed - 内容操作系统

## 概述

Seed 是 0-1 项目的"内容引擎"，从工作日志中自动生成多平台发布内容。

## 核心流程

```
工作日志 (memory/YYYY-MM-DD.md)
    ↓
Seed 读取 + 分析
    ↓
内容生成引擎
    ├── 📝 文字帖子 (多平台)
    ├── 🖼️ 配图建议
    └── 🎬 视频脚本 (数字人)
```

## 内容生成规则

### 输入触发
- 用户触发：告诉我今天/这周做了什么
- 自动触发：cron 定期检查新内容

### 内容丰富度判断
- **丰富**：拆成 3-5 天发布，细水长流
- **一般**：合并精华，单次集中发布
- **贫瘠**：仅生成一句话更新 + 待补充标记

### 输出格式

#### 1. 文字帖子
```markdown
## [平台] 标题

正文内容...

#标签1 #标签2
```

#### 2. 配图建议
```markdown
## 图片建议

- 类型：截图/概念图/数据图
- 构图：
- 色调：
```

#### 3. 视频脚本
```markdown
## 视频脚本 (数字人)

### 开场 (0-3s)
[数字人]：...

### 正文 (3-60s)
[数字人]：...

### 结尾 (60-63s)
[数字人]：...
```

## 目录结构

```
seed/
├── content/
│   ├── raw/          # 原始素材
│   ├── draft/        # 草稿
│   └── published/   # 已发布
├── assets/          # 素材库
├── queue/           # 发布队列
└── scripts/
    └── generate.sh  # 生成脚本
```

## 多语种支持

- 中文 (主力)
- English (可选)

## 数字人集成

视频脚本生成后，配合数字人工具生成视频：
- 工具：Seedance / 可灵 / 其他
- 输入：用户照片 + 脚本
- 输出：数字人说话视频

## 使用方式

用户触发 → `告诉我今天/这周做了什么` → 我生成内容
