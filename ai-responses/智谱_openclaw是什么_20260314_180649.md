## 回复内容
OpenClaw 是一个开源、本地优先的 AI Agent（智能体）平台，它不仅能聊天，还能在你自己的电脑/服务器上"真正做事"——比如读邮件、操作文件、浏览器自动化、执行脚本等，通过 WhatsApp、Telegram、Discord、微信、钉钉、飞书等聊天入口来指挥它，被很多人当成"会干活的数字员工"或"个人 AI 助手"。

核心架构：Gateway / Agent / Skills / Memory
- Gateway：网关，解析意图、路由
- Agent：智能体
- Skills：技能
- Memory：记忆
